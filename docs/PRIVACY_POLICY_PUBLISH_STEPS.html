隐私政策公网发布（最短流程）

方案 A：GitHub Pages（推荐，免费）

  1. 新建仓库或使用当前仓库。
  2. 在仓库创建文件 privacy-policy.html。
  3. 将 docs/PRIVACY_POLICY_WEB.md 内容转成 HTML 放入该文件。
  4. 进入仓库 Settings -> Pages。
  5. Source 选择 Deploy from a branch，分支选 main，目录选 /(root) 或 /docs。
  6. 保存后等待 1-3 分钟，得到公开 URL：
    https://<your-username>.github.io/<repo>/privacy-policy.html
  7. 打开 URL 自检：是否无需登录即可访问、移动端可打开。

方案 B：官网静态页

  1. 在你官网新增 /privacy 页面。
  2. 粘贴 docs/PRIVACY_POLICY_WEB.md 内容。
  3. 确保全站 HTTPS。
  4. 打开无痕窗口检查可访问性。

提交到 Chrome Web Store 前检查

  1. CWS 的 Privacy policy 字段填写公网 URL，而不是仓库相对路径。
  2. 页面包含更新时间（Last updated）。
  3. 页面包含联系邮箱（support@blockbot-extension.com 或你的真实支持邮箱）。
  4. 页面内容与扩展真实行为一致（第三方模型、后台标签页、本地存储）。
